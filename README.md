## 📌 Overview
The Football Club Management System is a comprehensive desktop application built using Core Java. It is designed to streamline the complex administration of teams, players, and coaching staff within a football club. By digitizing club operations, this system reduces administrative overhead and ensures accurate, real-time tracking of team rosters and entity management.

## ✨ Key Features
* **Dynamic Player Transfers:** Engineered a robust onboarding and transfer module to seamlessly handle complex roster updates, player acquisitions, and departures.
* **Entity Management:** Centralized administration for all club personnel, including players, managers, and coaching staff.
* **Team Administration:** Organize and track team lineups, staff assignments, and hierarchical club structures.
* **Core Java Architecture:** Built entirely on Core Java principles, emphasizing object-oriented design, encapsulation, and modularity.
